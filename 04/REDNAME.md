## 배운 것 정리

- 데이터셋 분할: `torch.randn`으로 가상 데이터를 만들고, `train_test_split`을 두 번 사용해 학습/검증/테스트를 60/20/20으로 나누는 방법을 배웠다. `random_state`를 쓰면 항상 같은 방식으로 섞인다는 것도 알게 됐다.

- K-NN: "나랑 가장 가까운 K명한테 물어봐서 다수결로 결정"하는 알고리즘이다. K값이 너무 작으면 예민하고 너무 크면 둔감해진다는 것을 배웠다.

- 4가지 알고리즘 비교: Perceptron, SVM, Random Forest, Naive Bayes를 동일한 데이터에 적용해봤다. 데이터가 100개일 때와 1000개일 때 결과가 다르게 나왔고, 데이터 양이 모델 성능에 영향을 준다는 걸 직접 확인했다.

- Data Augmentation: 기존 학습 데이터에 노이즈를 추가해서 데이터를 2배로 늘려봤다. 데이터가 이미 충분하거나 단순한 경우엔 오히려 성능이 떨어질 수 있다는 걸 배웠다. (증강 전 98.0% → 증강 후 96.5%)

- 활성화 함수: Sigmoid, Tanh, ReLU, Leaky ReLU를 직접 정의하고 그래프로 시각화했다. ReLU는 음수를 완전히 0으로 만들고, Leaky ReLU는 음수도 0.1배로 살려둔다는 차이를 그래프로 직접 확인했다.

- MLP: `make_moons`로 비선형 데이터를 만들고, 파이토치로 다층 퍼셉트론을 설계해 학습시켰다. 입력층→은닉층→출력층 구조로 99% 정확도를 달성했다.

- CNN: MNIST 손글씨 숫자 이미지 60000장으로 CNN을 학습시켜 98.83% 정확도를 달성했다. Conv2d로 이미지를 필터로 훑고, MaxPool2d로 크기를 줄이는 구조를 이해했다.
